Block Element Modifier Beispiel

How to use

BEM - Block Element Modifier ist eine Methode, mit der Sie wiederverwendbare Komponenten und Code-Sharing in der Front-End-Entwicklung erstellen können. Hier können Sie erfahren, wie Sie die BEM-Methode verwenden können.

Einführung

BEM ist eine hoch effiziente, stake und simple Methode der Namenskonvenzion, mitder der Front-End Code einfacher zu lesen und zu verstehen ist. So ist es auch einfacher, mit dem Code zu Arbeiten und ihn zu erweitern. Der Code ist so robuster und expliziter, das sorgt auch für mehr strenge.

Naming

Der BEM-Ansatz stellt sicher, dass jeder, der an der Entwicklung einer Website beteiligt ist, mit einer einzigen Codebasis arbeitet und dieselbe Sprache spricht. Durch die korrekte Benennungskonvention von BEM können Sie sich besser auf Änderungen an Ihrer Website vorbereiten
